Környezeti változó beállítása Csak akkor szükséges, ha az Android Studio első indításánál hibaüzenetet jelent meg!

Ha az Android Studio első indításánál hiányzik a JAVA_PATH környezeti változó, akkor sajnos a Java JDK telepítése folyamán elmulasztotta megmondani a Windowsnak, hogy merre találja legfontosabb fájljait. Ezt a mulasztást most nekünk kell jóvá tennünk.

Két lehetőség közül is választhatunk:

  1. Windows 7/8 alatt
    Töltsd le és futtasd RENDSZERGAZDAKÉNT az alábbi fájlt: JavaHome.bat
    Sikeres futtatás után az alábbi ablakot látod:
  2. Manuálisan Kézileg is hozzá lehet adni az új változót a "Rendszertulajdonságok" dialóguson keresztül.
    1. Start menü
    2. jobb gombbal Számítógép
    3. tulajdonságok
    4. Csak Windows 7/8 alatt: Speciális rendszerbeállítások (baloldalt keresd!)
    5. A megjelenő ablakban a Speciális fül legyen kiválasztva
    6. Alul Környezeti változók gomb, majd az alsó Új… gombra kattintva meg kell mondani, hogy hol található a feltelepített java fejlesztői kit (JDK):
      A „Változó neve” legyen JAVA_HOME a „Változó értéke” pedig ahova a JDK-t telepítetted (ez valószínűleg a következő: C:\Program Files\Java\jdk1.8.0_20).
    7. Végül mindegyik ablakot zárd be az OK gombbal, és próbáld meg újra elindítani az Android Studiot!
