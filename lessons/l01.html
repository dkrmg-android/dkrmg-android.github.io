Mielőtt látványos és hasznos Android programokat írhatnánk, első lépésként fel kell telepítenünk az Android programozáshoz szükséges programokat a számítógépre. A feladat elég egyszerű, ennek ellenére igyekeztünk kimerítően részletes leírást adni. Ha mindeközben mégis elakadnál, nyugodtan írj egy emailt (dkrmg.android@gmail.com), segítünk.

A következő programokra lesz szükségünk (ebben a sorrendben !!!):

  • Java JDK
  • Android Studio

Java JDK telepítése

  1. Töltsd le ingyenesen a JDK legújabb verzióját a gyártó honlapjáról (160-170MB): http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
    • Ehhez el kell fogadnod a licensz feltételeket
    • Ki kell választanod, hogy 32 (x86) bites vagy 64 bites (x64) Windows van a számítógépeden.

  2. Indítsd el a letöltött .exe fájlt, majd a dialógus segítségével a "Next" végül a "Close" gomb ismételt megnyomásával telepítsd a programot.

    Ha a géped korábban nem találkozott volna a Java semmilyen változatával, akkor települni fog a JRE (Java Runtime Environment) is

    A Windows XP panaszkodhat telepítés végén, hogy az operációs rendszer elavult a Java ezen verziójához. Ez esetünkben nem fog gondot okozni, nyugodtan kattinthatunk az OK gombra.

Android studio telepítése

  1. Töltsd le ingyenesen a Google honlapjáról http://developer.android.com/sdk/installing/studio.html (362 MB)
    • Ehhez kattints a "Download Android Studio..." gombra
    • Fogadd el a felhasználói feltételeket
    • Kattints a lap alján található letöltés gombra

  2. Telepítés közben Ellenőrizzük, hogy a telepítés helye (Destination Folder) ne tartalmazzon szóközöket!

    Ha az alapértelmezett hely mégis tartalmazna szóközt, telepíts inkább pl. a "C:\Android" mappába.

    Windows XP-n az alapértelmezett útvonalban szerepel az "Application Data" mappa, amely sajnos tartalmaz egy szóközt! Ezért XP-n hozzunk létre pl a C: meghajtó gyökerében egy mappát (pl.: AndroidSzakkor), ezen belül egy android-studio nevűt, és ide telepítsünk!

  3. Telepítés után az Android Studio üdvözöl minket a következő képernyővel

    1. Válaszd az alsó opciót (ez az első telepítésed, az Android Studio hiába szeretne, nem tud semmit importálni)
    2. Kattints az OK gombra

    Ha ehelyett az alábbi hibaüzenettel találkoznál, a számítógépedet szemmel láthatóan ma morcos hangulatában találtad (érzékeny lények).

    Sajnos ez egy gyakori hiba, amire megoldást itt találsz. Utána gyere ide vissza :)

  4. Végre elindult az Android Studio, ahol a jövőben kiváló "app"-okat írhatunk okostelefonjainkra. Már csak néhány apró lépés, és tényleg készen állunk.

    1. Kattints a Configure -> SDK Manager menüpontra

    2. A listában győződj meg arról, hogy semmi nincs kipipálva, csak az alábbi dolgok:

      • Android 4.4.2 (API 19)
        • SDK Platform
        • Intelx86 Atom System Image
        • Sources for Android
      • Extras
        • Android Support repository
        • Android Support Library
        • Google Repository
        • Google USB Driver
        • Intel x86 Emulator Accelerator

    3. Kattints az "Install X packages..." gombra, és fogadd el a licensz feltételeket. Ezt a lista mindhárom fő pontján meg kell tenned (amíg az összes pipa zöld nem lesz, mint a képernyőn). Utána "Install"

Most már tényleg készen állunk az első programunk elkészítésére!
